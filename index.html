📌 학생의 질문

"이차방정식 x² + 5x + 6 = 0을 어떻게 풀어?"

💡 Claude의 답변

이 문제를 인수분해로 풀어보겠습니다:

Step 1: 6을 두 수의 곱으로 나타내기 (합이 5)

→ 2 × 3 = 6, 2 + 3 = 5 ✓

Step 2: 인수분해

(x + 2)(x + 3) = 0

Step 3: 해 구하기

x = -2 또는 x = -3

📌 학생의 코드 (에러 포함)

def calculate_average(numbers):
    total = sum(numbers)
    average = total / len(numbers)  # 빈 리스트면 에러!
    return average

result = calculate_average([])  # 실행하면 ZeroDivisionError

💡 Claude의 피드백

문제: 빈 리스트가 들어오면 len(numbers) = 0이 되어 0으로 나누기 에러 발생

해결책: 리스트 길이 확인

def calculate_average(numbers):
    if not numbers:
        return 0  # 또는 None, 또는 에러 메시지
    return sum(numbers) / len(numbers)

📌 학생의 초안

"Climate change is very bad. It make the weather hot. We need to do something about it."

💡 Claude의 개선 제안

개선된 버전:

"Climate change poses unprecedented challenges to ecosystems and human societies worldwide. Rising global temperatures disrupt weather patterns, leading to increasingly severe droughts, floods, and hurricanes. Addressing this crisis requires immediate, coordinated action from individuals, businesses, and governments."

개선 사항:

  • 구체적인 용어 사용 (very bad → unprecedented challenges)
  • 문법 수정 (make → poses)
  • 예시 추가 (droughts, floods, hurricanes)
  • 문장 구조 개선
