# Claude 교육용 한 페이지 소개 사이트 설계

**작성일**: 2026-06-20  
**대상**: 학생  
**목적**: Claude의 기능과 능력을 설명하는 교육용 페이지

---

## 개요

학생들을 위한 Claude 소개 웹사이트. 한 페이지 구성으로 Claude가 무엇인지, 어떤 일을 할 수 있는지, 실제 사용 사례를 쉽게 이해할 수 있도록 설계.

---

## 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 마크업 | HTML5 | 정적 페이지, 프레임워크 불필요 |
| 스타일 | CSS3 | 순수 CSS, 의존성 최소화 |
| 스크립트 | 최소한의 JavaScript | 스크롤 효과, 상호작용만 필요 |
| 호스팅 | GitHub Pages | 자동 배포, 비용 무료 |
| 배포 | 자동 (gh-pages branch) | 푸시 시 자동 배포 |

---

## 페이지 구조 & 레이아웃

### 전체 레이아웃
```
┌─────────────────────────────────────┐
│         Header (20%)                │
│  "Claude 배우기" + 짧은 소개        │
├─────────────────────────────────────┤
│   Section 1: Claude란? (20%)        │
│  AI 어시스턴트 정의, Anthropic 소개 │
├─────────────────────────────────────┤
│ Section 2: 할 수 있는 일들 (25%)    │
│     4개 기능 카드 (코딩, 글쓰기     │
│        분석, 학습)                  │
├─────────────────────────────────────┤
│  Section 3: 실제 예시 (25%)         │
│     학생 관심 예시 2-3개            │
├─────────────────────────────────────┤
│   Section 4: 시작하기 (10%)         │
│  접근 방법 (웹, API, 요금)          │
├─────────────────────────────────────┤
│          Footer                     │
│   링크, 저작권, 소셜 등             │
└─────────────────────────────────────┘
```

### 반응형 디자인
- **Desktop (1024px+)**: 전체 섹션 한눈에, 카드 그리드
- **Tablet (768px-1023px)**: 섹션 가로 스크롤, 카드 2열
- **Mobile (< 768px)**: 전체 세로 스크롤, 카드 1열

## 각 섹션 상세 설계

### 1. Header (네비게이션 + 히어로)
**콘텐츠**:
- 로고/제목: "Claude 배우기" (또는 "Claude는 뭘까?")
- 부제: "학생을 위한 AI 어시스턴트 완벽 가이드"
- 배경: 그라데이션 (파란색~보라색)
- CTA 버튼: "아래로 스크롤" 또는 "시작하기"

**기술**:
- 고정 높이 (뷰포트 100vh)
- 센터 정렬
- 텍스트 컨트라스트 높음

---

### 2. Section 1: Claude란? (20%)
**제목**: "Claude는 무엇인가요?"

**콘텐츠** (글:이미지 = 60:40):
- 텍스트:
  - "Claude는 Anthropic이 만든 AI 어시스턴트입니다"
  - "자연어로 질문하면, 코딩, 글쓰기, 분석, 학습 등 다양한 작업을 도와줍니다"
  - "학생들이 더 빠르고 더 똑똑하게 배우는 데 도움을 줍니다"
- 이미지: Anthropic 로고 또는 Claude 마스코트

**기술**:
- Flexbox 레이아웃
- 텍스트 정렬: 왼쪽 또는 중앙

---

### 3. Section 2: 할 수 있는 일들 (25%)
**제목**: "Claude로 뭘 할 수 있을까?"

**카드형 레이아웃** (4개 카드, 그리드):
각 카드는 아이콘 + 제목 + 짧은 설명

| 카드 | 제목 | 설명 | 아이콘 |
|------|------|------|--------|
| 1 | 코딩 도우미 | 버그 수정, 코드 이해, 프로젝트 조언 | 💻 |
| 2 | 글쓰기 | 에세이, 리포트, 창의적 글쓰기 도움 | 📝 |
| 3 | 학습 | 수학, 과학, 역사 개념 설명 | 🎓 |
| 4 | 분석 | 데이터 이해, 문제 해결 논리 | 🔍 |

**기술**:
- CSS Grid (2x2)
- 호버 효과: 카드 살짝 올라가기, 그림자 추가
- 반응형: 모바일은 1열

---

### 4. Section 3: 실제 예시 (25%)
**제목**: "실제로 어떻게 쓸까?"

**3개 예시** (스크롤 또는 탭형):
1. **예시 1: 수학 숙제**
   - 학생이 물어본 질문: "이차방정식을 어떻게 풀어?"
   - Claude 답변: 단계별 설명 (스크린샷 또는 텍스트)

2. **예시 2: 파이썬 코드 디버깅**
   - 학생이 제출한 코드 (에러 포함)
   - Claude 피드백: 뭐가 잘못되었고, 어떻게 고칠지

3. **예시 3: 영어 에세이**
   - 학생 초안
   - Claude 피드백: 문법, 구조, 아이디어 개선

**기술**:
- 탭 또는 아코디언 형식 (모바일 친화적)
- 코드는 하이라이트 (코드 블록 스타일)
- Before/After 비교

---

### 5. Section 4: 시작하기 (10%)
**제목**: "지금 바로 시작하세요"

**콘텐츠**:
- 3가지 접근 방법:
  1. **웹**: claude.ai 방문 (무료, 가입만 하면 됨)
  2. **API**: 개발자라면 API 활용 (요금 정보)
  3. **학교 계정**: 학교 이메일로 특별 혜택

**기술**:
- 버튼 또는 링크
- 외부 링크는 새 탭에서 열기

## 디자인 스타일 가이드

### 색상
- **Primary**: #4B5AFF (파란색)
- **Secondary**: #8B5CF6 (보라색)
- **Background**: #FFFFFF 또는 #F8F9FA
- **Text**: #1F2937 (어두운 회색)
- **Accent**: #06B6D4 (청록색, 강조용)

### 폰트
- **Heading**: Google Fonts "Inter" 또는 시스템 폰트 (bold, 24-48px)
- **Body**: Google Fonts "Inter" 또는 시스템 폰트 (regular, 16-18px)
- **Code**: Monospace (JetBrains Mono 또는 시스템)

### 간격
- 섹션 간 패딩: 60-80px (세로)
- 카드 간 간격: 20-30px
- 텍스트 줄 높이: 1.6

### 효과
- 호버: 카드 올라가기, 배경색 변경
- 스크롤: 부드러운 스크롤 (smooth scroll)
- 트랜지션: 0.3초
